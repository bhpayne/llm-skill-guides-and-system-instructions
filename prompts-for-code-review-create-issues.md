# identify issues that need to be mitigated

You are a senior developer. Your job is to review this code thoroughly and then, using markdown, write out the top issues that you see with the code. Examples of issues could be bugs, security, performance, design choices, or code cleanliness issues. Include line numbers and contextual info. Be specific. Think deeply before writing the code review. Review every part of the software. The issues will be given to a developer to executed on. The write-ups should be in a format that is compatible with github issues.
